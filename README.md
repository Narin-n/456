# Мамылхаб — статический сайт

Этот репозиторий содержит простую статическую страницу для компьютерного клуба "Мамылхаб".

Файлы:
- `index.html` — главная страница
- `css/styles.css` — стили
- `js/main.js` — простая логика: модалка бронирования и плавный скролл

Как запустить:
1. Откройте `index.html` в любом браузере.

Настройка контактов и часов работы:
- В `index.html` найдите блок с id="contacts" и замените номер телефона и адрес.

Дальше можно:
- Подключить форму бронирования к вашему бэкенду или сервису (например, Telegram, Google Forms или собственный API).
- Добавить реальные фотографии в папку `assets` и заменить заглушки в галерее.
- Вставить карту (Google/OSM) в блок карты.

Если нужно, могу подготовить версию с формой, привязанной к Telegram или с серверной отправкой (укажите предпочтения).
